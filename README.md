> Projetos do curso **Profissão Programador**, reconstruídos com arquitetura moderna (serverless, deploy 1-clique na Vercel) em vez de servidores dedicados.

## Rodar localmente

```bash
cd <projeto>      # ex: youtube-clone
npm install
npm run dev
```

Cada projeto é independente. Os clones (Next.js) buildam com `npm run build`; o Batman Tickets é estático.
